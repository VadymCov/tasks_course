# __init__ в python

Метод __init__ в Python — це спеціальний (конструктор) метод класу, який автоматично викликається під час створення нового об'єкта цього класу. Він використовується для ініціалізації атрибутів об'єкта, тобто для задання початкових значень змінних, які будуть належати цьому об'єкту.

Основні властивості методу __init__:

- Викликається автоматично при створенні екземпляра класу.
- Має як перший параметр self, який посилається на сам об'єкт.
- Приймає додаткові параметри, що використовуються для ініціалізації атрибутів об'єкта.
- Використовується для налаштування стану об'єкта одразу після його створення.

Приклад використання:

```python
class MyClass:
    def __init__(self, param1, param2):
        self.param1 = param1
        self.param2 = param2

об'єкт = MyClass(10, 20)
```

У цьому прикладі при створенні об'єкта MyClass викликається метод __init__, який задає значення атрибутів param1 і param2 для цього об'єкту.

Отже, __init__ — це основний спосіб в Python встановити початкові значення властивостей новоствореного об'єкта і підготувати його до роботи.
