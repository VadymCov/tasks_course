

Метод __iter__ у Python робить об'єкт ітерованим (iterable). Це спеціальний метод, який дозволяє вашому класу працювати з циклами for та іншими конструкціями, які очікують ітеровані об'єкти.
Основна функція
__iter__ повинен повертати ітератор — об'єкт, який має метод __next__(). Найчастіше метод просто повертає self, якщо клас також реалізує __next__.
Простий приклад
class Counter:
def __init__(self, max_value):
self.max_value = max_value
self.current = 0

    def __iter__(self):
        return self  # повертає сам себе як ітератор
    
    def __next__(self):
        if self.current < self.max_value:
            self.current += 1
            return self.current
        else:
            raise StopIteration  # сигналізує про кінець ітерації
    
# Використання

counter = Counter(5)
for num in counter:
print(num)  \# виведе: 1, 2, 3, 4, 5

Коли використовується
Метод __iter__ автоматично викликається, коли ви:
Використовуєте об'єкт у циклі for
Передаєте об'єкт у list(), tuple(), set()
Використовуєте розпакування: a, b, c = obj
Застосовуєте функції як sum(), map(), filter()
Завдяки __iter__ ваші власні класи можуть поводитися як вбудовані колекції Python (списки, словники тощо).

Метод `__iter__` у Python робить об'єкт ітерованим (iterable). Це спеціальний метод, який дозволяє вашому класу працювати з циклами `for` та іншими конструкціями, які очікують ітеровані об'єкти.

### Основна функція

Метод `__iter__` повинен повертати ітератор — об'єкт, який має метод `__next__()`. Найчастіше метод просто повертає `self`, якщо клас також реалізує `__next__`.

### Приклад коду

```python
class Counter:
    def __init__(self, max_value):
        self.max_value = max_value
        self.current = 0
    
    def __iter__(self):
        return self  # повертає сам себе як ітератор
    
    def __next__(self):
        if self.current < self.max_value:
            self.current += 1
            return self.current
        else:
            raise StopIteration  # сигналізує про кінець ітерації

# Використання
counter = Counter(5)
for num in counter:
    print(num)  # виведе: 1, 2, 3, 4, 5
```


### Коли використовується метод `__iter__`

Метод автоматично викликається, коли ви:

- Використовуєте об'єкт у циклі `for`
- Передаєте об'єкт у функції `list()`, `tuple()`, `set()`
- Використовуєте розпакування, наприклад: `a, b, c = obj`
- Застосовуєте функції як `sum()`, `map()`, `filter()`

Завдяки реалізації `__iter__` ваші власні класи можуть поводитися як вбудовані колекції Python (списки, словники тощо).
