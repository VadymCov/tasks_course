## Set
**Set** — це вбудований тип контейнера, що представляє незмінну множину унікальних хешованих елементів. Множини реалізовані як хеш-таблиці, тому операції перевірки приналежності, додавання і видалення елементів у середньому виконуються за O(1).

Ключові факти :
- Елементи множини унікальні; дублікати видаляються автоматично.
- Множини невпорядковані — не підтримують індексацію.
- Допускаються лише хешовані об'єкти (наприклад, числа, рядки, кортежі); не з хешовані (списки, словники, множини) — ні.
- Підтримуються стандартні операції над множинами: об'єднання (union), перетин (intersection), різниця (difference), симетрична різниця (symmetric_difference).
- Існує також тип frozenset — незмінна (immutable) множина, яка може використовуватись як ключ у словнику.

Короткий приклад (офіційний стиль):
```python
s = {1, 2, 3}
s.add(4)                 # додати елемент
s.remove(2)              # видалити (підніме KeyError, якщо немає)
s.discard(10)            # видалення без помилки, якщо відсутній
u = s.union({3, 5})      # об'єднання
i = s.intersection({2,3})# перетин
```
