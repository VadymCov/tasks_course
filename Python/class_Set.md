**Set** у Python — це невпорядкована колекція унікальних елементів.

### Коли і для чого використовують
- Видалення дублікатів зі списків.
- Тести на приналежність (перевірка, чи є елемент) — дуже швидко.
- Операції над множинами: об'єднання, перетин, різниця, симетрична різниця.
- Приклад використання у алгоритмах: пошук унікальних значень, перевірка пересічення множин (наприклад, спільних елементів у двох списках).

### Основні властивості
- Елементи унікальні (дублікати автоматично видаляються).
- Невпорядковані — немає індексів, не можна звертатись за позицією.
- Можуть містити лише хешовані типи (наприклад, числа, рядки, кортежі), не можуть містити списки чи словники.

### Створення та основні операції — приклади коду
```python
# створення
s = set([1, 2, 3, 2])   # s == {1, 2, 3}
s2 = {3, 4, 5}

# перевірка приналежності
if 2 in s:
    print("є 2")

# додати/видалити
s.add(4)        # {1,2,3,4}
s.remove(1)     # якщо 1 немає — помилка
s.discard(10)   # якщо нема — без помилки

# операції над множинами
union = s | s2          # {2,3,4,5}
intersection = s & s2   # {3,4}
difference = s - s2     # елементи в s, яких немає в s2
sym_diff = s ^ s2       # елементи в одному з них, але не в обох

# довжина
len(s)  # кількість елементів
```
